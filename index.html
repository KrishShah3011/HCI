Doodle Transformation

Stable Diffusion ControlNet PyTorch

Engineered an Image-to-Image Transformation pipeline that generates high-fidelity images from user-provided scribbles.

  • Built the complete Stable Diffusion architecture from scratch.
  • Trained the model to generate high-quality, conditioned image outputs.
  • Implemented the ControlNet (Scribble Condition) architecture from scratch.
  • Integrated ControlNet with the diffusion backbone for precise, guided image generation.
  • Integrated a previously developed Image Captioning Model to provide descriptive textual explanations of generated images.

Chroma Sight

PyTorch Transformers YOLO CNNs

Engineered a comprehensive, multi-model solution to assist visually impaired individuals by providing real-time environmental context.

  • Image Captioning: Constructed a model using Vision Transformer (ViT) architecture to generate descriptive explanations of images.
  • Depth Perception: Designed models employing ViT and Stereo Vision to provide users with a sense of object depth and distance.
  • Traffic Light Detection: Implemented a YOLO Object Detection model with 92.27% accuracy.
  • Integrated all three models to create a seamless and comprehensive user experience.
